Faculty Landing (Section) Page

Summary

The faculty landing pages also follow a three column layout, similar to the homepage. The landing pages have an optional section description that occupies 100% of the page width. The section description should tell the user what type of information the user can expect to find on the page. After the intro text there is a divider followed by three columns: The section subpages, secondary content area, and sidebar.

View landing page layout wireframe

Intro Text (Optional)

Summary

The flexible messaging area is used to convey important messages relating to the content and goals of the page. Special consideration needs to be taken when selecting images, as they need to work with the content on all devices.

Properties
  • Width: 320px - 1170px
  • Color: #6A7870
  • Font Size: 16px
  • Character Limit: ~300 characters (no more than 3 lines of text on large displays)
Allowed Components
  • Text

Section Page Listing (First Column)

Summary

The section page listing is in the primary column, and contains all of the pages underneath the current section, and a very brief description of what information the page has.

Properties
  • Width: 320px - 580px
  • Height: Depends on content
Allowed Components
  • Subpage List

Secondary Content Area (Second Column)

Summary

The secondary content area is a space to put content that is not crucial to the page but is related to the page's topic/goals and serves a purpose to the user.

Properties
  • Width: 320px - 722px
  • Height: Depends on content
Allowed Components
  • Feed
  • Video
  • Rich Text
  • Custom HTML
  • Inline Form
  • Data List

Sidebar (Third Column)

Summary

The sidebar area is reserved for content that is not directly related to the page's content, but may provide of some use to the user.

Properties
  • Width: 170px - 670px
  • Height: Depends on content
Allowed Components
  • Any Sidebar Module
