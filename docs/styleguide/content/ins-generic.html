Institutional Generic Page

Summary

The generic layout is used for the few pages within the institutional site that are used to show basic content. The layout consists of the breadcrumb, page title, content area, and sidebar. When building a page using the generic layout, consider utilizing the grid system to build columns and keep the line length under ~100 characters.

Primary Content Area

Summary

The primary content area is designated for content that will contribute to achieving the goals that have been set for the page.

Properties
  • Width: 320px - 940px
  • Height: Depends on content
Allowed Components
  • Rich Text
  • Custom HTML

Sidebar

Summary

The sidebar area is reserved for content that is not directly related to the page's content, but may provide of some use to the user.

Properties
  • Width: 170px - 670px
  • Height: Depends on content
Allowed Components
  • Any Sidebar Module
