Images

Images should be used sparingly on the page, and only when they compliment the content. Images that are poor quality, obvious stock images, excessively large, or distasteful are not to be used.

Images change size based on the container they are in. The image size also will adjust based on the device that it is viewed on. When choosing an image to use make sure it is an appropriate size and aspect ratio to be displayed on all devices.

Images should not be used to display text content, only photos and figures. Any text required should be placed over the image in HTML so it remains accessible and searchable.

Image Frame

Wrapping an image in a div with the "frame" class will add a white border with rounded corners and a shadow to an image. Also, wrapping the image in an "image-shadow" gives the image an inset appearance.

<div class="frame">
  <a class="image-shadow"><img src="#url" /></a>
</div>
With a Caption

Captions are encouraged on images to give a short summary of the image, or if the image is a link, what the user can expect to see when they click on it.

<div class="frame">
  <div class="image-caption-container">
    <a class="image-shadow">
      <img src="http://www.placehold.it/500x290" />
    </a>
    <p class="caption">
      This is a sample caption for an image with a frame.
    </p>
  </div>
</div>

This is a sample caption for an image with a frame.

Inset Image

Images can also be given an inset appearance without a frame, using the "image-shadow" class. Adding the "rounded" class will add rounded corners.

<div class="span4">
  <a class="image-shadow rounded"><img src="#url" /></a>
</div>
