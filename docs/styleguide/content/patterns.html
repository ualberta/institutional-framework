Patterns

Extruded Container

The extruded container is used to to bring elements out of the page. The extruded container is used to hold images, videos, program types, sidebar modules, and more. Use the .extrude class for content pieces, and the .frame class for images and videos.

Example
Container
Markup
<div class="extrude">Container</div>

Inset Container

The inset container gives the area the look that it is sunk into the page to draw the eye. The inset container is used primarily for feature areas.

Example
Container
Markup
<div class="inset">Container</div>

Lists of Links

Lists of links that occur in the main content area of the page should be contained in a data list. If there is a link to view more of whatever is listed, then the bottom-row class should be utilized.

Example
  • Costs, Tuition, and Fees
  • Major Deadlines
  • Registrar
  • View the Calendar
Markup
<ul class="data-list">
    <li><a href="#">Costs, Tuition, and Fees</a></li>
    <li><a href="#">Major Deadlines</a></li>
    <li><a href="#">Registrar </a></li>
    <li class="bottom-row"><a href="#">View the Calendar</a></li>
</ul>

Lists of Links with Descriptions

This pattern is most commonly used with news articles or events, where the link is the article or event name and the text that follows is a brief description of what the user can expect to find if they click the link. Similar to the list of links, the .bottom-row class can be used to provide a link to view more.

Example

  • Title Link One

    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Vivamus malesuada convallis fermentum. Sed semper gravida leo et porttitor.

  • Title Link Two

    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Vivamus malesuada convallis fermentum. Sed semper gravida leo et porttitor.

  • View More
Markup
<ul class="news-list data-list">
	<li class="news-row">
		<div class="news-description">
			<h4><a href="#">Title Link</a></h4>
			<p class="news-teaser">Teaser</p>
		</div>
	</li>
	<li class="bottom-row"><a href="#">View More</a></li>
</ul>

Media Items

The media item pattern consists of a thumbnail image, heading link, date, and description. The pattern can be used to display news items, posts by users, events, videos, or similar content.

Example

  • Title One

    Posted on February 28, 2013

    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Vivamus malesuada convallis fermentum.

  • Title Two

    Posted on February 28, 2013

    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Vivamus malesuada convallis fermentum.

Markup
<ul class="news-list data-list">
	<li class="news-row small-image">
  	<div class="frame news-thumbnail">
    	<a href="#" class="image-shadow">
      	<img src="http://placehold.it/81x78">
      </a>
    </div>
    <div class="news-description">
    	<h4><a href="#">Title</a></h4>
      <span class="news-date">Date</span>
      <p class="news-teaser">Description</p>
    </div>
  </li>
</ul>
