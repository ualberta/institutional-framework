Faculty Layouts

Summary

The Faculty layouts are very similar to the Institutional layouts, but have the following differences:

  • Displays the University logo in the blade area instead of the banner area
  • Displays only the faculty name in the banner
  • Has a faculty specific footer
  • Has dropdown menus
  • Does not have audience navigation
