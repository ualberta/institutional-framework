Breadcrumb

Summary

The breadcrumb shows the path to the current page the user is on. This facilities navigation throughout the website and allows the user to learn the structure of the website. The breadcrumb is placed between the global navigation and page title.

Components

  • Path: The breadcrumb consists of one line that shows the path to the current page. Every parent page is listed, linked, and separated by a forward slash all the way back to the homepage.
