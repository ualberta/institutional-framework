Dropdown Menus

Summary

Dropdown menus are used to display deeper level pages in the information architecture so the user can find what they are looking for more quickly. There is also a content section inside of the dropdown menu that can be used to provide content pieces that may be useful.

Components

  • Sub-Navigation: The pages under the parent section are displayed in the sub-navigation. The number of sub-pages should range from 3 to 7.
  • Content Space: The content space should be used to display a content piece related to the parent item. Functional content should be placed here not static images or marketing text.

Devices

Mobile

Does not display on mobile devices, only the top level navigation is displayed.

Tablet

Dropdown menus appear when a user selects an item in the global navigation.

Desktop

Dropdown menus display when hovering over an item in the global navigation.
