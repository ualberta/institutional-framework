Grid

You can utilize the grid system that is built into the framework to arrange content in various ways that may better suit the content you are displaying.

The grid system consists of 12 fluid columns fit into a container. To use the grid, create a row and then create the columns you desire within the row. You can use the preset span classes (.span1, .span2, .span3, etc) to make an element span multiple columns. The total number of columns spanned per row should be 12. The grid behaves differently at different resolutions.

Mobile

  • Row Width: 100%
  • Columns: 1
  • Column Width: 100%
  • Gutters: none

Tablet

  • Row Width: 100%
  • Columns: 12
  • Column Width: ~5.80%
  • Gutters: ~2.76%

Desktop

  • Row Width: 980px
  • Columns: 12
  • Column Width: ~6.38%
  • Gutters: ~2.13%

High Resolution

  • Row Width: 1170px
  • Columns: 12
  • Column Width: ~5.98%
  • Gutters: ~2.56%

Creating Rows and Columns

The following example shows one way three rows with different columns can be built. The first row contains 3 equally sized columns that each span 4 rows. The second row contains two columns that span 6 rows. The last row contains two columns, one of which spans 8 columns, and one which spans 4 columns.

Example
.span4 .span4 .span4
.span6 .span6
.span8 .span4
Markup
<div class="row-fluid">
	<div class="span4">.span4</div>
	<div class="span4">.span4</div>
	<div class="span4">.span4</div>
</div>
<div class="row-fluid">
	<div class="span6">.span6</div>
	<div class="span6">.span6</div>
</div>
<div class="row-fluid">
	<div class="span8">.span8</div>
	<div class="span4">.span4</div>
</div>

Separating Rows

You can use a horizontal rule element to separate rows. This is useful when switching topics or presenting a new section. Using the "soften" class gives the line a faded look on the edges. Adding the "double" class will add margins to both the top and bottom of the divider. The divider will occupy the entire width of its container.
