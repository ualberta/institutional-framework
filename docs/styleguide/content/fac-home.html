Faculty Homepage

Summary

The faculty homepage follows a three column layout, similar to the institutional homepage layout. The first row spans the entire width of the page and consists of the Feature Messaging Area and also the Audience Area. Following the first row there are three columns: Primary content area, secondary content area, and the sidebar. Faculty pages can have up to six navigation items and must contain the 4 standardized navigation items: About, Programs, Departments, Research.

View homepage layout wireframe

Flexible Messaging Area

Summary

The flexible messaging area is used to convey important messages about upcoming events, deadlines, and activities happening at the University. Special consideration needs to be taken when selecting images, as images need to properly display content on all devices.

Properties
  • Width: 320px - 570px
  • Height: 350px
Allowed Components
  • Custom HTML
  • Image w/ Caption
  • Video
  • Carousel

Audience Area

Summary

The Audience Area is a space that has been set aside for the faculty to target specific audiences and provide information that they would be interested in. A tabbed component is most frequently used to target multiple audiences. Only content that moves the user further in the goal path should be placed here. Not static images or unaccompanied "facts" that does not progress the user towards the intended goal.

Properties
  • Width: 320px - 570px
  • Height: 350px
Allowed Components
  • Custom HTML
  • Tab Component

Primary Content Area

Summary

The primary content area is designated for important information that will appeal to a wide audience. This space is most commonly devoted to faculty specific news headlines and teasers.

Properties
  • Width: 320px - 580px
  • Height: Depends on content
Allowed Components
  • News headlines w/ teaser

Secondary Content Area

Summary

The secondary content area is a space to put content that is not crucial to the page but serves some purpose to the user. This is good area to highlight an event, upcoming deadlines, recruitment related content, etc.

Properties
  • Width: 320px - 722px
  • Height: Depends on content
Allowed Components
  • Video
  • Custom HTML
  • Rich Text

Sidebar

Summary

The sidebar area is reserved for content that is not directly related to the page's content, but may provide of some use to the user.

Properties
  • Width: 170px - 670px
  • Height: Depends on content
Allowed Components
  • Twitter Feed
  • Link Buckets (RSS, Blogs, etc)
  • Custom HTML
  • Any Sidebar Module
