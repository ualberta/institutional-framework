Sidebar Items

Summary

Items in the sidebar are used to display content that is related to the content on the current page. It is a place to put content that the user may find useful. Items that appear in the sidebar have similar formatting consisting of a container, with a header, and a content space.

Components

  • Heading: The heading should be a short phrase that accurately represents the content that will appear in the content space. A link can also be specified that the heading will link to on mobile devices.
  • Icon: The icon is optional, and should only be used if there is a suitable icon that accurately represents the content.
  • Content Space: Content space can support data lists, images, rich text, and more. Only content that serves a purpose should be placed in this area.

Devices

Mobile

The content space is collapsed, and can be expanded by tapping on the icon or row space. If a link has been specified, clicking on the heading will take the user to that link.

Tablet

The sidebar is moved up in the page on the right side and all content spaces are visible.

Desktop

Sidebar items appear on the right and all content spaces are visible.
