Faculty Generic Page

Summary

The generic layout is used for the few pages within the site that are used to show basic content. The generic faculty layout is the most flexible. If your page has sibling third level pages, or subpages, the left navigation will be displayed. Another optional page element that is available are sidebar modules, which will transform the page into a 3 column layout (Left navigation, content area, sidebar).

View generic page layout wireframe

Left Navigation

Summary

Left navigation shows the sibling and child pages of the current page the user is viewing. Left navigation is auto populated based on your website structure. External links should not be placed in the left navigation, instead external links should be placed in the page content or in a related links sidebar module.

Properties
  • Width: 270px - 320px
  • Height: Depends on content

Content Area

Summary

The content area is designated for the content that will contribute to achieving the goals that have been set for the page.

Properties
  • Width: 320px - 1170px
  • Height: Depends on content
Allowed Components
  • Rich Text
  • Custom HTML
  • Custom Apps

Sidebar

Summary

The sidebar area is reserved for content that is not directly related to the page's content, but may provide of some use to the user.

Properties
  • Width: 170px - 670px
  • Height: Depends on content
Allowed Components
  • Any Sidebar Module
