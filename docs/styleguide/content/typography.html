Typography

Headings

Headings are used in the page content to facilitate scanning and identifying content that follows the heading. Note that using h1 and h2 in the page content is discouraged unless they are wrapped in a section or article tag. h1 and h2 are reserved for Site and Page title, so in page content you should start with h3. Any text in a heading tag should use title casing.

  • Heading One <h1>
  • Heading Two <h2>
  • Heading Three <h3>
  • Heading Four <h4>
  • Heading Five <h5>

Links

Links in the content area of the page should always be in the Primary Green (#007c41). To help prevent the links from getting lost in large paragraphs of text, an underline should be used. This will apply to content heavy pages, news articles, and other similar page types. A button should be used for important links that need more attention brought to them.

Paragraphs

Paragraphs use the default text color, and should ideally not exceed 80 characters per line to facilitate readability. There is a intro-text class that can be used for a larger text size to emphasize a certain piece of text. This is useful for news teasers and summary paragraphs. A lighter secondary text paragraph can be used for pieces of information that are not critical such as dates or teasers in data lists.

  • Standard Paragraph <p>
  • Introduction Paragraph <p class="intro-text">
  • Secondary Paragraph <p class="secondary-text">

Lists

There are a variety of lists should be used when displaying choices, events, news, definitions, steps, links or any type of data where common sense dictates to use one.

  • Unordered List <ul>

    Default list for content when orering is not important.

  • Ordered List <ol>

    Default ordered list, used for outlining a list of steps or other times when numbering is important.

  • Definition List <dl>

    List used for words and their definitions

  • Unstyled List <ul class="unstyled">

    Use for standard lists where no bullet or marker is required.

  • Data List <ul class="data-list">

    Typically used for lists of links, but can also be used for text, events, and media items.
