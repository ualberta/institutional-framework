Headings

Headings are used in the page content to facilitate scanning and identifying content that follows the heading. Note that using h1 and h2 in the page content is discouraged unless they are wrapped in a section or article tag. h1 and h2 are reserved for Site and Page title, so in page content you should start with h3.

  • Heading One <h1>
  • Heading Two <h2>
  • Heading Three <h3>
  • Heading Four <h4>
  • Heading Five <h5>
  • Heading Six <h6>

Links

Links in the content area of the page should always be in the Primary Green (#007c41). To help prevent the links from getting lost in large paragraphs of text, an underline should be used. This will apply to content heavy pages, news articles, and other similar page types. A button should be used for important links that need more attention brought to them.
