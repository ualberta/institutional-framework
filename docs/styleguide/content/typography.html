Typography

Content Headings

Headings are used in the page content to facilitate scanning and identifying content that follows the heading.

  • Only use Heading 2 through Heading 6 (h2 - h6)
    Heading 1 (h1) is reserved for the page title for SEO purposes.
  • Use title casing
    e.g. Here is Some Text Inside a Heading Tag

Links

Standard links are green by default. Links appearing within content pieces like paragraphs, news articles, etc. will have underlines.

  • Use descriptive text
    Do not use phrases like "click here" for links.
  • Use the data list for small link collections
    3 to 7 links
  • Use the filterable link collection module for categorized collections of links.
    3 to 7 categories, 3+ links per category
  • Use the accordion module for categorized collections of links.
    7+ categories, 3+ links per category

  • Secondary Paragraph <p class="secondary-text">

Lists

There are a variety of lists should be used when displaying choices, events, news, definitions, steps, links or any type of data where common sense dictates to use one.

  • Unordered List <ul>

    Default list for content when orering is not important.

  • Ordered List <ol>

    Default ordered list, used for outlining a list of steps or other times when numbering is important.

  • Definition List <dl>

    List used for words and their definitions

  • Unstyled List <ul class="unstyled">

    Use for standard lists where no bullet or marker is required.

  • Data List <ul class="data-list">

    Typically used for lists of links, but can also be used for text, events, and media items.
