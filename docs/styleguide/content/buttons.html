Buttons

Buttons are used to indicate to the user that they can perform an action. Text inside of buttons should follow title casing.

Primary Calls to Action

There are three buttons that have been designated as our "Primary Calls to Action". Buttons that represent one of these calls to action have unique styling reserved for them, and should not be used for other buttons.

Standard Buttons

The standard button is to be used for generic calls to action and other areas where buttons are required. Buttons should be used for performing an action, and not just for standard links.

Sizes
  • .btn-large: Used for campaign work and very rare conditions where a large button is deemed necessary.
  • .btn-small: Small buttons are the standard size, and the default for form submit buttons.
  • .btn-micro: Micro sized buttons should be used within containers when space is limited.
Colors
  • .btn-grey: Grey buttons should be used by default.
  • .btn-green: Green buttons are reserved for feature messaging areas and other areas where extra attention needs to be added to the button.
